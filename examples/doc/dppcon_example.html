Example program for dppcon

This example estimates the condition number in the 1-norm (or ∞-norm) of the matrix A, where
A= 4.16 -3.12 0.56 -0.10 -3.12 5.03 -0.83 1.18 0.56 -0.83 0.76 0.34 -0.10 1.18 0.34 1.18 .  
Here A is symmetric positive definite, stored in packed form, and must first be factorized by dpptrf. The true condition number in the 1-norm is 97.32.
