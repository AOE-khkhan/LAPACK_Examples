A QR factorization is performed on the first 4 rows of A using zgeqrt after which the first 4 rows of B are updated by applying QT using zgemqrt. The remaining row is added by performing a QR update using ztpqrt; B is updated by applying the new QT using ztpmqrt; the solution is finally obtained by triangular solve using R from the updated QR.
